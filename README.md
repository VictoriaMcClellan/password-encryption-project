# Password Encryption Project

This project demonstrates encryption and decryption of a password file using three methods:

- Caesar Cipher
- Vigenere Cipher
- Advanced Encryption Standard (AES)

## Files Included
encrypt_passwords.py – Python encryption program  
passwd.txt – Password input file  
Encryption_Assignment_APA.docx – APA research paper  

## How to Run

Install Python and run:

python encrypt_passwords.py

## Purpose

This project compares classical cryptography (Caesar and Vigenere ciphers) with modern encryption (AES).
